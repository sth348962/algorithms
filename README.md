# Описание

Набор различных алгоритмов и структур данных на PHP.

# Установка и запуск

## Установка

```
git clone https://github.com/sth348962/algorithms
cd algorithms
composer install
```

## Тестирование

```
composer test-unit-run
```

## Создание нового теста

```
composer test-unit-generate
```
